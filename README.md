# 🎓 Student Performance Predictor

A modular Machine Learning project that predicts student exam performance based on academic and demographic features using multiple regression models.

---

## 🚀 Overview
This project uses multiple regression algorithms to predict student scores based on features like study hours, previous scores, and other academic factors.  
It follows a **modular ML pipeline structure** for better scalability and maintainability.

---

## 🧩 Project Modules
- Data Ingestion  
- Data Transformation  
- Model Training  
- Model Evaluation  
- Prediction Pipeline  
- Flask Web Application  

---

## ⚙️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Flask  
- HTML/CSS  

---

## 🧠 Models Used
- Linear Regression  
- Decision Tree Regressor  
- Random Forest Regressor  
- XGBoost Regressor  
- Gradient Boosting Regressor  

---

## 📊 Result
- Best Model R² Score: ~0.88
